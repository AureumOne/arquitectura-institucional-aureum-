## Propósito de esta documentación

Esta documentación describe una **infraestructura institucional** diseñada para coordinar custodia de oro físico mediante **terceros independientes**, bajo un marco explícito de gobernanza y evidencia verificable.

Su alcance es intencionalmente limitado.

**Esto no es:**

* un producto de inversión
* una plataforma de distribución
* un custodio de activos
* un broker ni intermediario financiero
* una propuesta de retorno o rendimiento

**Esto es:**

* una arquitectura institucional
* una capa de gobernanza y control
* un sistema de coordinación basado en evidencia
* una estructura diseñada para reducir ambigüedad fiduciaria

No se solicita participación.
No se solicita confianza.
No se sugiere expectativa alguna.

Este material existe exclusivamente para **análisis institucional y revisión técnica**.
