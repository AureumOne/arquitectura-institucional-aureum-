## Por qué existe esta infraestructura

Esta infraestructura fue diseñada para reducir ambigüedad, no para generar creencia.

Resuena con quienes valoran:

* claridad estructural
* límites explícitos
* realidad verificable

Quienes esperan:

* garantías
* promesas
* narrativas simplificadas

no son el público objetivo.

> **La arquitectura no pide ser confiada.
> Pide ser correctamente comprendida.**
